# プロンプト例

## メモから下書きする

```text
verifiable-minimal-writing skill を使ってください。
下のメモを [文書種別] にしてください。
読み手: [読み手]
読み手の目的: [目的]
足りない事実を捏造しないでください。
確認済みの事実、仮定、不明点を分けてください。
下書き後に、削除または弱めた根拠なし主張の短いレビューNoteを付けてください。

メモ:
...
```

## 既存文章をレビューする

```text
verifiable-minimal-writing skill を使って、この下書きをレビューしてください。
返すもの:
1. 根拠がない、または自信が強すぎる主張
2. 冗長な文または段落
3. 隠れた不明点
4. 構造上の問題
5. 圧縮した改訂版

下書き:
...
```

## 判断から ADR を作る

```text
verifiable-minimal-writing skill を使って ADR を作ってください。
決定: ...
確認済みの事実: ...
検討した代替案: ...
不明点: ...
架空の文脈を追加しないでください。
```

## 安全に圧縮する

```text
verifiable-minimal-writing skill を使って、これを約30%短くしてください。
制約、根拠、不明点、エッジケース、トレードオフは消さないでください。
圧縮後に主張が不正確になる場合は、長い版を残してください。
```

## skill 自体を改善する

```text
verifiable-minimal-writing skill を使って、この失敗ケースを分析してください。
失敗を分類し、ルール変更を提案し、ADR を書き、eval case を追加してください。

失敗ケース:
...
```
