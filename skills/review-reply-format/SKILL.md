# AIレビュー返信フォーマット

## Overview

codex・AI自動レビューへのコメント返信を事務的・統一フォーマットで行う。丁寧語なし、体言止め、指摘引用→対応内容の構造に固定する。

## フォーマット

```
> [指摘の該当箇所や要点を引用]

- <対応内容1（体言止め）>
- <対応内容2>
- <除外したもの＋理由>
```

## ルール

- **体言止め** — 丁寧語（「〜しました」「ありがとうございます」等）は使わない
- **指摘を引用してから返す** — 引用なしに対応内容だけを書かない
- **除外した場合は理由を明記** — 対応しない指摘は無視せず、理由を添える

## やってはいけないこと

- 「検討します」「ご指摘ありがとうございます」などの丁寧語を使う
- 引用なしに返信する
- 対応/非対応の理由を書かずにスキップする

## 例

```
> `get_foo()` の戻り値が型ヒントと合っていない

- 戻り値の型を `Optional[Foo]` に修正
- 呼び出し側の None チェックも追加

> テストケースが正常系のみ

- 異常系（存在しない ID）のケースを追加
- 権限エラー系は別 PR で対応予定（スコープ外のため）
```
